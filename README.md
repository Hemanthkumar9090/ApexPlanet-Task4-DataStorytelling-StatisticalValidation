# 📊 ApexPlanet-Task4-DataStorytelling-StatisticalValidation

## 📌 Project Overview

This project is part of the **ApexPlanet Data Analytics Internship – Task 4**. The objective is to transform raw sales data into meaningful business insights through **data storytelling**, **exploratory data analysis (EDA)**, **statistical validation**, and **business recommendations**.

The project demonstrates how data-driven insights can support better business decision-making by combining visual analytics with hypothesis testing.

---

## 🎯 Objectives

- Analyze sales data to uncover business insights.
- Create compelling data stories using visualizations.
- Perform statistical hypothesis testing.
- Validate findings using appropriate statistical methods.
- Present actionable business recommendations.

## 📌 Hypothesis Testing

### Hypothesis 1

**Null Hypothesis (H₀):**

There is no significant difference in spending between male and female customers.

**Alternative Hypothesis (H₁):**

There is a significant difference in spending between male and female customers.

**Statistical Test Used**

- Independent Sample T-Test

---

### Hypothesis 2

**Null Hypothesis (H₀):**

Gender and Product Category are independent.

**Alternative Hypothesis (H₁):**

Gender and Product Category are associated.

**Statistical Test Used**

- Chi-Square Test
